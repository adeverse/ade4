# The ade4 package

This package is developed in the Biometry and Evolutionary Biology Lab
(UMR CNRS 5558) - University Lyon 1. It contains Data Analysis functions
to analyse Ecological and Environmental data in the framework of
Euclidean Exploratory methods, hence the name ade4.  

ade4 is characterized by (1) the implementation of graphical and
statistical functions, (2) the availability of numerical data, (3) the
redaction of technical and thematic documentation and (4) the inclusion
of bibliographic references.  

To cite ade4, please use `citation("ade4")`.

## Author

Stéphane Dray, Anne-Béatrice Dufour, and Jean Thioulouse. Contributions
from Daniel Borcard, Stéphanie Bougeard, Thibaut Jombart, Pierre
Legendre, Jean R. Lobry, Sébastien Ollier, Sandrine Pavoine and Aurélie
Siberchicot. Based on earlier work by Daniel Chessel.
